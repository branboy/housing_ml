## Dataset

Due to size constraints, the dataset is provided as a downloadable ZIP file.

### 📦 Download

[Download Dataset]()

### 📂 Setup Instructions

1. Download the ZIP file from the link above
2. Extract it into the **root directory** of the project

After extraction, your folder structure should look like:

data/
├── raw/
├── processed/
└── images/

outputs/
├── models/
└── predictions/

### ⚠️ Notes

* Do **not** rename folders
* Ensure the `data/` and `outputs/` directories are at the project root
* Large files are excluded from version control
