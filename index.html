Overview

Abstract

This project presents a multi-signal housing price prediction system that stacks three complementary signals: (1) an XGBoost structured model trained on 2.2 million US homes across 54 states, (2) a RentCast Automated Valuation Model (AVM) cross-checked with geography-aware blending weights, and (3) a CLIP zero-shot visual condition scorer that interprets uploaded property photos. The structured model achieves a 28.3% median absolute percentage error (MAPE) on a held-out test split of 8,445 homes. An outlier detection mechanism identifies when the structured model undershoots premium markets and applies a graduated premium boost of up to +30%. When tested on a known sold home ($931,000 in Olney, MD), the full pipeline predicted $929,592 — 0.15% error. The system is deployed as a Streamlit web application supporting structured inputs, optional street address for AVM enrichment, and photo uploads for condition scoring.

Outlier Detection

Premium Market Override

When AVM and market stats both diverge from the structured model by >42% log-space while agreeing with each other, the structured model weight drops to 5% and a premium boost of up to +30% is applied — critical for luxury non-CA markets.

Geography-Aware

CA vs. Non-CA Weights

California uses a conservative AVM blend (10–40%) due to rich structured training data. Non-CA markets receive higher AVM weights (25–60%) because the model has thinner state-specific learning outside California.

Methodology

How It Works

Signal 1

XGBoost Structured Model

Trained on 2.2M homes across 54 states on log(1 + price). 40+ engineered features across four groups:

  • Geographic — city/zip target encoding, state ordinal
  • Structural — sqft, beds, baths, ratios, sqft², flags
  • Property — age, decade bucket, property type (5 cats)
  • Extras — lot size, HOA, pool, garage, stories

max_depth=5 · min_child_weight=15 · CUDA-accelerated (~5 min)

Signal 2

RentCast AVM Blend

When a street address is provided, the RentCast API returns a property-level AVM estimate with confidence level (High / Medium / Low / Unknown).

  • Low confidence → 10% blend weight
  • High confidence (non-CA) → 60% blend weight
  • Extra weight when AVM diverges >0.30 log-units
  • Zip market anchor corroborates AVM signal
Signal 3

CLIP Visual Condition Scoring

CLIP ViT-B/32 scores uploaded photos zero-shot — no labeled training data required.

  • 4 positive prompts (modern, renovated, high-end)
  • 4 negative prompts (dated, worn, deferred maintenance)
  • score = mean(pos sim) − mean(neg sim)
  • Adjustment capped at ±15% · up to 5 photos averaged
